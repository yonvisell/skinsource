# SkinSourceSim

SkinSourceSim is a portable static-browser workbench based on the SkinSource toolbox and dataset. It predicts dynamic tactile signals in the hand and arm by convolving user-defined vibration stimuli with SkinSource impulse-response measurements and superposing responses across selected input locations.

The app supports generated stimuli, WAV import with resampling, compact add/replace multi-input setup, interpolated or measured-sensor RMS surface visualization, selected-output time and frequency views with frequency-axis log toggles, session save/load, and local export of data, images, and short response videos.

Reference: N. Tummala et al., "SkinSource: A Dataset of Whole-Arm Skin Vibrations for Tactile Rendering", IEEE Haptics Symposium 2024, https://doi.org/10.1109/HAPTICS59260.2024.10520852

Original SkinSource toolbox: https://github.com/neelitummala/skinsource
