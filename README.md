# SkinSource 2.0

SkinSource 2.0 is a portable static-browser workbench based on the SkinSource toolbox and dataset. It predicts dynamic tactile signals in the hand and arm by convolving user-defined vibration stimuli with SkinSource impulse-response measurements and superposing responses across selected input locations.

It is a local-first browser application: data and computation remain in the browser. It facilitates rapid haptic validation without requiring new physical experiments.

The app opens with an active default response and supports editable generated or WAV input signals, compact add/replace multi-input setup, interpolated or measured-sensor RMS surface visualization, stacked or overlaid selected-output time and frequency views, frequency-axis log toggles, session save/load, and local export of data, images, and short response videos.

Reference: N. Tummala et al., "SkinSource: A Data-Driven Toolbox for Predicting Touch-Elicited Vibrations...", IEEE Haptics Symp. 2024, https://doi.org/10.1109/HAPTICS59260.2024.10520852

Original SkinSource toolbox: https://github.com/neelitummala/skinsource
